# Learning TypeScript 🚀

Welcome to my TypeScript learning repository! This project serves as a personal sandbox, documentation hub, and code laboratory where I have been learning, practicing, and mastering the fundamentals and advanced concepts of TypeScript.

The goal of this repository is to transition from standard JavaScript to writing type-safe, robust, and scalable code.

---

## 🛠️ Tech Stack & Tools

* **Language:** TypeScript
* **Runtime/Environment:** Node.js
* **Compiler:** `tsc` (TypeScript Compiler)
* **Linter/Formatter:** ESLint & Prettier (Optional: adjust if you used them)

---

## 📚 What I've Learned So Far

Here is a breakdown of the core TypeScript concepts covered in this repository:

### 1. Basic Types & Type Inference
* Primitive Types (`string`, `number`, `boolean`, `null`, `undefined`)
* Arrays and Tuples
* Enums (`enum`)
* `any`, `unknown`, and `void`

### 2. Interfaces & Type Aliases
* Defining object shapes using `interface` and `type`
* Extending interfaces vs. Intersecting types (`&`)
* Optional properties (`?`) and Readonly properties (`readonly`)

### 3. Functions
* Typing function arguments and return values
* Arrow functions and optional parameters
* Function overloading

### 4. Advanced Types
* Union (`|`) and Intersection Types
* Type Guards and Type Assertion (`as`)
* Literal Types

### 5. Generics
* Creating reusable components and functions using `<T>`
* Generic constraints

### 6. React + TypeScript (Optional - Remove if not applicable)
* Typing functional components (`React.FC` or direct props)
* Typing hooks like `useState` and `useEffect`
* Handling form events (`React.FormEvent`, `React.ChangeEvent`)
